## TROUBLESHOOTING

### Error: "ModuleNotFoundError: No module named 'flask'"
**Solusi**:
```powershell
pip install flask
```

### Error: "FileNotFoundError: jmlh_psn_rwt_jln_mnrt_klnk_brdsrkn_jk_11.xlsx"
**Solusi**:
- Pastikan file Excel ada di folder proyek
- Check nama file (case-sensitive)

### Error: Port 5000 sudah digunakan
**Solusi**:
Edit app.py, ganti port:
```python
app.run(debug=True, host='0.0.0.0', port=5001)
```

### Dashboard terbuka tapi gambar tidak muncul
**Solusi**:
- Jalankan data_visualization.py dulu
- Pastikan folder static/ ada
- Refresh browser (Ctrl+F5)

## TIPS & TRICKS

### Tip 1: Virtual Environment
Gunakan venv untuk isolasi dependencies:
```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

### Tip 2: Generate PDF Report
Install reportlab untuk export statistik ke PDF:
```powershell
pip install reportlab
```

### Tip 3: Backup Data
Selalu backup:
- data_cleaned.csv
- Semua visualisasi di static/
- statistics_report.txt

### Tip 4: Performance
Jika dataset besar (>100k rows):
- Sampling untuk quick analysis
- Caching untuk repeated queries
- Use SQLite untuk better performance
