LANGKAH 2: DEPLOY KE PYTHONANYWHERE
================================================================================

A. PERSIAPAN PYTHONANYWHERE
----------------------------
1. Daftar di https://www.pythonanywhere.com (Free Account)
2. Verifikasi email
3. Login ke dashboard PythonAnywhere

B. CLONE REPOSITORY DARI GITHUB
--------------------------------
1. Di dashboard PythonAnywhere, buka tab "Consoles"
2. Klik "Bash" untuk membuka Bash console
3. Jalankan perintah berikut:

   cd ~
   git clone https://github.com/USERNAME/jkn-dashboard.git
   cd jkn-dashboard
   ls -lh

   GANTI "USERNAME" dengan username GitHub Anda!

C. SETUP VIRTUAL ENVIRONMENT
-----------------------------
1. Masih di Bash console, jalankan:

   python3.10 -m venv venv
   source venv/bin/activate
   pip install --upgrade pip
   pip install -r requirements.txt

   CATATAN: Proses instalasi membutuhkan waktu 2-5 menit

D. KONFIGURASI WEB APP
-----------------------
1. Tutup Bash console
2. Di dashboard, buka tab "Web"
3. Klik "Add a new web app"
4. Klik "Next" untuk domain default (USERNAME.pythonanywhere.com)
5. Pilih "Manual configuration"
6. Pilih "Python 3.10"
7. Klik "Next"

E. KONFIGURASI WSGI FILE
-------------------------
1. Di halaman Web app, scroll ke section "Code"
2. Klik link "WSGI configuration file"
3. HAPUS semua isi file, lalu ganti dengan:

# ============================================================================
import sys
import os

# Path ke project
path = '/home/USERNAME/jkn-dashboard'
if path not in sys.path:
    sys.path.insert(0, path)

# Path ke virtual environment

activate_this = os.path.join(venv_path, 'bin', 'activate_this.py')

# Activate virtual environment

    exec(file_.read(), dict(__file__=activate_this))

from app import app as application
# ============================================================================

   GANTI "USERNAME" dengan username PythonAnywhere Anda (2 tempat)!

4. Klik "Save" (Ctrl+S atau tombol di atas)

================================================================================
TROUBLESHOOTING
================================================================================

ERROR 1: "502 Bad Gateway"
---------------------------
Solusi:
- Periksa WSGI file, pastikan path sudah benar
- Cek error log di tab "Web" â†’ "Error log"
- Pastikan virtual environment sudah terinstall dengan benar

ERROR 2: "ModuleNotFoundError: No module named 'flask'"
-------------------------------------------------------
Solusi:
- Buka Bash console
- cd ~/jkn-dashboard
- source venv/bin/activate
- pip install -r requirements.txt

ERROR 3: "FileNotFoundError: data_cleaned.csv"
----------------------------------------------
Solusi:
- Pastikan file data_cleaned.csv ada di folder project
- Cek di Bash console: ls -lh ~/jkn-dashboard/

ERROR 4: Static files tidak muncul (gambar tidak muncul)
---------------------------------------------------------
Solusi:
- Periksa konfigurasi Static files di tab "Web"
- Pastikan path benar: /home/USERNAME/jkn-dashboard/static

ERROR 5: "Disk quota exceeded"
-------------------------------
Solusi:
- Project ini hanya 2 MB, seharusnya tidak ada masalah
- Jika tetap error, hapus file lain di akun PythonAnywhere
- Periksa penggunaan disk di tab "Files"
