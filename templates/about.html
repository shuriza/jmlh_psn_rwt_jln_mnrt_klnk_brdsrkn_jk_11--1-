Tentang Proyek

Informasi lengkap tentang dashboard dan metodologi

Deskripsi Proyek

Dashboard ini dibuat sebagai bagian dari Ujian Tengah Semester Ilmu Data dengan fokus pada visualisasi dan analisis data peserta rawat jalan berdasarkan program Jaminan Kesehatan Nasional (JKN).

Proyek ini mencakup seluruh proses data science pipeline mulai dari pengumpulan data, preprocessing, analisis statistik, hingga visualisasi interaktif berbasis web.

Tentang Dataset

Sumber Data

Dataset: Jumlah Peserta Rawat Jalan Menurut Kelompok Berdasarkan JKN

Dataset ini berisi informasi mengenai jumlah peserta program JKN yang melakukan rawat jalan, dikelompokkan berdasarkan berbagai kategori demografis dan geografis.

Metodologi

Data Preprocessing
  • Missing Values: Identifikasi dan penanganan data yang hilang menggunakan median (numerik) dan modus (kategorikal)
  • Duplicate Data: Deteksi dan penghapusan data duplikat
  • Outliers: Identifikasi outliers menggunakan IQR method dan penanganan dengan capping technique
Analisis Statistik Deskriptif
  • Perhitungan measures of central tendency (mean, median, mode)
  • Measures of dispersion (variance, standard deviation, range, IQR)
  • Measures of shape (skewness, kurtosis)
  • Analisis korelasi antar variabel numerik
Visualisasi Data
  • Histogram dan boxplot untuk distribusi data
  • Bar chart untuk data kategorikal
  • Heatmap untuk matriks korelasi
  • Time series plot untuk analisis tren (jika ada data temporal)
  • Comparative charts untuk analisis perbandingan
  • Interactive plots menggunakan Plotly
Dashboard Development
  • Web framework menggunakan Flask
  • Frontend dengan Bootstrap 5 untuk responsive design
  • Visualisasi interaktif dengan Plotly.js
  • RESTful API untuk data endpoints

Deployment

Dashboard ini dirancang untuk di-deploy ke PythonAnywhere.com, sebuah platform cloud hosting untuk aplikasi Python.

Langkah Deployment ke PythonAnywhere:
  1. Upload semua file proyek ke PythonAnywhere
  2. Install dependencies dari requirements.txt
  3. Konfigurasi web app dengan WSGI file
  4. Set static files directory ke folder /static
  5. Reload web app dan dashboard siap digunakan
