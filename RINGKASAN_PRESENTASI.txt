5. FLASK DASHBOARD
================================================================================

1. Halaman Home (/)
   - Info dataset (rows, columns)
   - Status preprocessing
   - Tabel statistik deskriptif

   - Galeri 7 visualisasi
   - Navigasi mudah dengan cards

3. API Endpoints (untuk interaktivitas)
   - /api/data: Get data preview (100 rows)
   - /api/stats: Get statistics

- Interactive Visualization: Plotly 5.18.0
- Frontend: Bootstrap 5 (CSS framework)
